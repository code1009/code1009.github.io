# 마크다운 문법

##  3. 목록

### 3.1. 순서있는 목록(번호)
* 예제

```
1. 첫번째
2. 두번째
3. 세번째
```

* 미리보기

> 1. 첫번째
> 2. 두번째
> 3. 세번째



### 3.2. 순서없는 목록(제목 기호: `*`, `+`, `-` 지원)
* 예제

```
* 빨강
  * 녹색
    * 파랑

+ 빨강
  + 녹색
    + 파랑

- 빨강
  - 녹색
    - 파랑
```

* 미리보기

> * 빨강
>   * 녹색
>     * 파랑
> 
> + 빨강
>   + 녹색
>     + 파랑
> 
> - 빨강
>   - 녹색
>     - 파랑
